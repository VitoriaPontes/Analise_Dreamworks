# Análise de filmes de animação e aventura da Dreamworks

Esse projeto é voltado para a criação de um Data WareHouse na nuvem AWS, seguindo as etapas de ETL, e o desenvolvimento de um dashboard relacionando os dados desse tema. Foram utilizados Python, Docker e ferramentas da AWS Cloud, como S3, Athena e QuickSight.

## [Parte 1](https://github.com/VitoriaPontes/Compass-UOL-D-A/tree/main/Sprint%205/Desafio)

Definição dos objetivos da análise, criação da camada raw no bucket AWS e upload dos datasets que serão utilizados.

## [Parte 2](https://github.com/VitoriaPontes/Compass-UOL-D-A/tree/main/Sprint%206/Desafio)

Criação da camada trusted no bucket AWS, com o refinamento dos dados que estavam em raw.

## [Parte 3](https://github.com/VitoriaPontes/Compass-UOL-D-A/tree/main/Sprint%207/Desafio)

Modelagem do Data WareHouse e criação da camarada refined do bucket AWS, agora com os dados voltados para os objetivos da análise dos filmes.

## [Parte 4](https://github.com/VitoriaPontes/Compass-UOL-D-A/tree/main/Sprint%208/Desafio)

Criação do dashboard com insights e storytelling sobre os filmes de animação e aventura da Dreamworks.
